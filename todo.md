## 📝 참고사항
- 각 Phase는 독립적으로 진행 가능하도록 설계
- Supabase는 문의하기와 커뮤니티 기능에만 사용
- 인증 없는 간단한 커뮤니티 시스템으로 구현
- 관리자 기능은 별도 인증 없이 간단한 인터페이스로 구현
- **개발 서버**: `npm run dev` 실행 후 `http://localhost:3000`에서 확인 가능
- **환경변수**: `.env` 파일 생성 필요 (README.md 참조)
